# 概要

このプロジェクトはキーボードコンポーネントの試作を目的としています。

# 機能要件

## 入力タイプ

以下の4種類の入力ボックスを用意します：

1. **数値型** - 数字のみ入力可能
2. **ひらがな** - ひらがな文字入力
3. **カタカナ** - カタカナ文字入力
4. **アルファベット** - 英字入力

## キーボードパターン

### パターン1: リアルタイム入力モード

- 入力項目フォーカス時にキーボードを表示
- キー入力が即座に入力項目へ反映
- 入力完了時にキーボードを自動的に閉じる

### パターン2: 確定入力モード

- 入力項目フォーカス時にキーボードを表示
- プレビューエリアに現在の入力値を表示
- キー入力でプレビュー値を編集（追加・削除）
- 「確定」ボタン押下でプレビュー値を入力項目へ反映
- 入力完了時にキーボードを自動的に閉じる
